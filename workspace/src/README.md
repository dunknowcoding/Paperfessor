# src/

The undergraduate's coding folder (per the user spec). Contents:

- `code/` - Python source for the project
- `datasets/` - datasets downloaded + preprocessed
- `tools/` - third-party tools and binaries
- `tests/` - unit and integration tests

Note: this is the UG's runtime workspace, not the framework's
top-level Python package (which is also called `src/`).
