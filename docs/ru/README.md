Paperfessor — исследовательский ассистент из трёх агентов, работающий на
вашей машине с вашим собственным API-ключом. Дайте ему направление
исследования (достаточно одной фразы) — и группа агентов работает как
маленькая лаборатория:

| Агент | Роль | API статуса |
|---|---|---|
| 🎓 **Аспирант** | Придумывает метод, раздаёт задачи, контролирует, пишет и проверяет статью | `planning / dispatching / monitoring / reviewing / writing / archiving` |
| 📚 **Магистрант** | Широкий поиск литературы (arXiv + OpenAlex + Scholar), строгое чтение полных текстов, извлечение доказательств | `websearch / reading / analyzing / reporting / idle / stopped` |
| 💻 **Бакалавр** | Реализует метод по строгому контракту, скачивает и готовит настоящие данные, запускает эксперименты с k сидов | `coding / thinking / reporting / idle / stopped` |

Каждое число в статье **измерено, а не выдумано**: датасеты — настоящие
публичные загрузки (загрузчики отвергают синтетические заглушки),
предложенный метод проверяется реальным запуском, а каждая страница PDF
проходит автоматическую проверку вёрстки, прежде чем запуск будет принят.

## Установка

```bash
# Python 3.11+
pip install -e ".[gui]"        # из клона
# или после публикации:
pip install paperfessor[gui]
```

Для PDF рекомендуется LaTeX (TeX Live/MiKTeX с `acmart`); без него —
запасной вариант `.docx` (pandoc) или Markdown.

## Первая настройка

API-ключи хранятся в **системном хранилище ключей** — никогда на диске, в
логах или в статье.

```bash
paperfessor key set minimax --key "sk-..."   # также openai / anthropic / google
paperfessor key test minimax                 # проверка связки
paperfessor models list                      # доступные модели
```

## Получить статью

```bash
paperfessor run "anomaly detection in multivariate time series"
```

Результаты в `workspace/`: `paper/body/paper.pdf` (PDF в формате
конференции), `src/results/results.json` (измеренные метрики, k = 3 сида,
среднее ± 95 % ДИ), настоящие рисунки и рабочие журналы агентов.
Нужно окно? `paperfessor-gui`.

## Ответственное использование и отказ от ответственности

Paperfessor создан **исключительно для исследовательских целей**.

- **Не** подавайте его результаты как собственную самостоятельную работу на
  конференции, в журналы или на учебные курсы; соблюдайте политику площадки
  в отношении ИИ-помощи, авторства и плагиата и раскрывайте участие ИИ, где
  это требуется.
- **Проверяйте всё.** Сгенерированные тексты, цитаты, код и числа должны
  быть проверены человеком перед любым реальным использованием.
- **Не** используйте инструмент для фабрикации исследований, манипуляции
  цитированием, «фабрик статей» и любых незаконных или обманных целей.

неправомерное использование этого ПО и за любые последствия его
применения.**
